# Vehicle Service Management System

Repository for the "Quản lý Hồ sơ & Bảo dưỡng Phương tiện (Garage)" capstone project.

## Structure

- `docs/`: Full project documentation from requirements to operations.
- `apps/`: Application code (`backend`, `frontend`).
- `packages/`: Shared packages and reusable modules.
- `infrastructure/`: Docker, database, migrations, and CI/CD related assets.
- `tests/`: Test suites (`unit`, `integration`, `e2e`, `performance`, `security`).
- `tools/`, `scripts/`: Development tools and automation scripts.

## Current status

Initial scaffold created. Content will be filled iteratively according to course deliverables.
